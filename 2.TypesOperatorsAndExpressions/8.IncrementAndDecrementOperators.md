# Increment And decerement Operators

The increment _++_ adds 1 to its operand, while the decrement operator _--_ subtracts 1.

_++_  and _--_ may be used either as prefix (before the variable as in _++n_) or as postfix (after the variable as in _n++_).
In both cases, the effect is to increment n . 
But the expression _++n_ increments n **before** its value is used, while _n++_ increments n after its value is used.

The increment and decrement operators apply only to variables; an expression like 
```c
(i + j)++
```
is **illegal**.
