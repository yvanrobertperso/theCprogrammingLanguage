# Declarations

All variables must be declared before use, although certain declarations can be made implicitly by content. 

A declaration specifies **a type** and containes a list of one or more variables of that type.

A variable can be initialized in its declaration. If the name is followed by an equal sign and an expression, the expression serves as an initalizer.

If the variable is not automatic, the initialization is done only once, cenceptionally before the program starts executing, and the initializer must be a constant expression.
An explicitly initialized automatic variable is initialized each time the function or block it is in us entered; the initializer may be any expression.

External and static variables are initialized to zero by default.

Automatic variables without explicit initializer have undefined (i.e. garbage) values.

The qualifier `const` can be applied to the declaration of any variable to specify that its value will not be changed.

```c
const double e = 2.71828182845905;
```

The `const` declaration can also be used with array arguments to indicate tha the function does not change that array.

```c
int strlen(const char[]);
```
