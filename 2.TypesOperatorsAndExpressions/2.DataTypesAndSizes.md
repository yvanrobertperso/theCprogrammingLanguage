# Data Types and Sizes.

There are only a few basic data types in C:
`char`  a single byte, capable of holding one character in the local character set
`int`   an integer, typically reflecting the natural size of integers on the host machine
`float` single-precision floating point
`double`double-precision floating point

In addition, there are number of qualifiers that can be applied to these basic types. 
`short` and `long` apply to integers:
```c
short int sh;
long inte counter;
```
The word `int`can be omitted, and often is.
The intent is to provide different lengths of integers, where parctical.
`short` and ints are at least 16 bits, `long`at least 32 bits, `short`is no longer than `int`, which is no longer than `long`.

The qualifier `signed` or `unsigned` may be applied to `char`or any integer.
`unsigned` numbers are always positive or zero and obey the law of arithmetic modulo $2^n$, where n is the number of bits in the type.
