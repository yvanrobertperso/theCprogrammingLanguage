# Variables and Arithmetic Expressions

See the program at /1.Introduction/fahr.c.

The line:
```c
/* print Fahrenheit-Celsius table */
```
is a comment.

In C, all variables must be declared before use, usually at the beginning of the function.

The `while` loop tests the condition in parentheses. If true, the loop body is executed. When false, execution continues after the loop.

Integer division truncates: any fractional part is discarded. For example, `5/9` would result in `0`.

```c
printf("%d\t%d\n", fahr, celsius);
```
Prints two integers separated by a tab.

`printf` is not part of the C language itself, but a standard library function.

Format specifiers:
- `%d`   : decimal integer
- `%6d` : decimal integer, at least 6 characters wide
- `%f`  : floating point
- `%6.2f`: floating point, at least 6 wide, 2 after decimal point

Others: `%o` (octal), `%x` (hex), `%c` (char), `%s` (string), `%%` (percent sign).

See also exercices/Introduction/fahr2.c for floating point numbers.
