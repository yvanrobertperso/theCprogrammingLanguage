# Characters Input and Output

A text stream is a sequence of characters divided into lines; each line ends with a newline character.

The standard library provides functions like `getchar` and `putchar`.

- `getchar()` reads the next input character.
- `putchar(c)` prints the character `c`.

See exercices/Introduction/fileCopy.c for an example.

The end of input is signaled by the value `EOF`, defined in `<stdio.h>`. Therefore, `c` is declared as `int`.

## Character Counting

See /Introduction/characterCount.c and /Introduction/characterCount2.c.

The operator `++` increments by one, `--` decrements by one. There are prefix and postfix forms.

`long` variables are at least 32 bits. `%ld` prints a `long` value.

## Line Counting

See /1.Introduction/lineCount.c.

The operator `==` tests for equality, `=` is assignment.

Character constants are in single quotes, e.g. `'A'`.
