# External Variables and Scope

See program /1.Introduction/longestLine.c as reference.

The variables in `main`, such as `longest`,`line`,etc., are private variables or local to `main`. Because they are declared within `main`, no other function can have direct access to them. 
Each local variable in a function comes to existence only when the function is called, and disappears when the fucntion is exited. This is why such variables are known as **automatic variables.**
As an alternative to automatic variables, it is possible to define variables that are external to all functions, that is variables that can be accessed by name by any function. 
An external variable must be defined, exactly once, outside of any function; this sets aside storage for it. The variable must also be declared in each function who wants to access it; this states the type of the variable.
