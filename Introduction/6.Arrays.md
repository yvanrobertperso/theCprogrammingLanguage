# Arrays

See the program `arrays.c`.

Declaration of an array:
```c
int ndigit[10];
```
Arrays in C always start at index 0.

To check if a character is a digit:
```c
if (c >= '0' && c <= '9')
```
The value `c - '0'` gives the digit as a number (0 to 9).

`char` variables are small integers and can be used like `int` in expressions.
