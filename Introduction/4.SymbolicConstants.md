# Symbolic Constants

It's bad practice to bury "magic numbers" like 300 or 20 in a program. It's better to give them meaningful names.

A `#define` line defines a symbolic name or constant:

```c
#define NAME REPLACEMENT
```

See `fahrSymbolic.c` for implementation.

Constants are conventionally written in uppercase. There is no semicolon at the end of a `#define` line.
