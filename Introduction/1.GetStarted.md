# Getting Started

The program `hello.c` shows the classic "hello world" program in C.

The file name must end with `.c`.

To compile, run:
```
$ gcc hello.c -o hello
```
This creates an executable file `hello`.

---

A C program consists of functions and variables. A function contains statements, variables store values.

In the example, the function is `main()`. Every C program must have a `main` function.

---

The line
```c
#include <stdio.h>
```
tells the compiler to include information about the standard input/output library.

Functions are called by name with arguments. Statements are enclosed in `{}`.

A string in double quotes, like `"hello, world\n"`, is a string constant. `\n` is a newline character.
