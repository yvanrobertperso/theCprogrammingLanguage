# Functions

See the program `power.c`.

A function definition has this form:

```c
return-type function-name(parameter declaration if any)
{
    declarations
    statements
}
```

The value that a function computes is returned by the `return` statement:
```c
return expression;
```

A function need not return a value.

Since `main` is a function like any other, it may return a value to its caller (typically 0 for normal termination).

A function prototype announces a function:
```c
int power(int base, int n);
```
Parameter names are optional in a function prototype, so the prototype could ahave been written:
```c
int power(int, int);
```
