# Character Arrays

See program longestLine.c

The functions `getline`and `copy`are declared at the beginning of the prgram, which we assume is one file.
`main`and `getline`communicate through a pair of arguments and a returned value. In `getline`, the arguments are declared by the line

```c
int getline(char s[], int lim);
```

which specifies that the argument `s` is an array and the second `lim`an integer. The purpose of supplying the size of an array in a declaration is to set aside storage.
Note that `getline`returns an integer value (the length of the line) but also fills the array (passed as the first argument) with the actual characters of the line.
The function `copy`is only used for its effect and returns no value. The return type of `copy`is `void`, which explicitly states that no value is returned.

`getline`puts the character '\0' (the null character, whose value is zero) at the end of the array it is creating to mark the end of the string of characters.
This conversion is also used by the C language: When a string constant like
```c
"hello\n"
```
appears in a C program, it is stored as an array of characters containing the characters in the string and terminated with a '\0' to mark the end.
